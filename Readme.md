### Создание среды разработки для Webpack.

Простой скрипт на python, который создает: 
- структуру приложения
- конфигурационные файлы
- файлы с примерами


Запуск

`python main`


Структура 

```
├── dist
├── js
├── main.py
├── node_modules
├── package.json
├── Readme.md
├── style
└── webpack.config.js

```

### Зависимости
- npm
- python +3.6

Скрипт запускал на Ubuntu 18.04. Полет нормальный.
